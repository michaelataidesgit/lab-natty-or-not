# Simulação Visual  

## Objetivo e Papel no Projeto

Esta simulação visual tem como objetivo traduzir o **Decision Copilot Framework** em uma representação gráfica clara, permitindo que líderes compreendam rapidamente:

- Como a decisão é estruturada
- Onde a IA entra no processo
- Onde a responsabilidade humana permanece
- Como a governança é aplicada

O visual foi concebido para apoiar:

- Comunicação executiva
- Alinhamento entre áreas
- Avaliação rápida por stakeholders não técnicos

A imagem não substitui o framework textual, mas o complementa, reduzindo ambiguidade e acelerando a compreensão.
