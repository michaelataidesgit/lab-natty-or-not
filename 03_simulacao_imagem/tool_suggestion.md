# Ferramentas Sugeridas para Criação Visual

O diagrama pode ser criado em qualquer ferramenta de visualização estratégica. As opções recomendadas são:

- Miro AI
- Whimsical AI
- FigJam

Critérios de escolha:

- Facilidade de leitura
- Clareza visual
- Capacidade de exportar imagem (PNG ou SVG)

A estética deve ser simples. A prioridade é clareza conceitual, não design sofisticado.
